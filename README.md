# Random Forest Regression and Classification

## Overview

This project implements machine learning models for both classification and regression tasks using Random Forest and other algorithms. It includes data preprocessing, model training, evaluation, and visualization.

## Evaluation Metrics

### Classification

* Accuracy
* Precision
* Recall
* F1 Score
* ROC-AUC

### Regression

* Mean Absolute Error (MAE)
* Mean Squared Error (MSE)
